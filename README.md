# Habit Tracker (Трекер звичок)
Простий веб-застосунок для відстеження щоденних корисних звичок.

## 🛠 Технічний опис
- **Frontend**: HTML5, CSS3 (Flexbox/Grid).
- **Логіка**: Vanilla JavaScript (робота з DOM та LocalStorage).
- **Структура**: Проєкт організований з розділенням контенту (HTML), стилів (CSS) та логіки (JS) у папці `src/`.

## 🚀 Інструкція для локального запуску
Щоб запустити проект локально, виконайте наступні кроки у вашому терміналі:

1. Клонуйте репозиторій:
```bash
git clone https://github.com/aniazablotska/habit-tracker.git
```
2. Перейдіть у папку проекту:
```bash
cd habit-tracker
```
3. Відкрийте головний файл у браузері:
```bash
start src/index.html
# або просто відкрийте файл вручну
```
